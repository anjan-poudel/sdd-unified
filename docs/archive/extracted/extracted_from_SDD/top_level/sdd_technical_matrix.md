# SDD V2 Technical Matrix

This document provides a comprehensive, at-a-glance technical reference for the Specification-Driven Development (SDD) V2 framework. It maps the features and tasks of the SDD workflow to the specific agents, commands, models, MCPs (Meta-Cognitive Primitives), and tools involved in each stage.

| Task/Feature    | Primary Agent         | User Command(s)                                      | Model                  | MCPs / Primitives                                                              | Key Tools                                                      |
| :-------------- | :-------------------- | :--------------------------------------------------- | :--------------------- | :----------------------------------------------------------------------------- | :------------------------------------------------------------- |
| **Define**      | `Business Analyst`    | `/feature create {description}`                      | `google/gemini-2.5-pro`    | `Requirements Elicitation`, `Problem Decomposition`                          | `write_to_file`                                                |
| **Design L1**   | `Architect`           | `/feature design --level 1`                          | `google/gemini-2.5-pro`    | `Architectural Pattern Selection`, `System Modeling`                         | `read_file`, `write_to_file`                                   |
| **Review L1**   | `User (as Reviewer)`  | `/feature review-design --level 1 {approve\|reject}` | `claude-3-sonnet`      | `Critical Analysis`, `Architectural Validation`                              | `read_file`                                                    |
| **Design L2**   | `Principal Engineer`  | `/feature design --level 2`                          | `claude-sonnet-4.5`    | `Component Breakdown`, `Service-Level Interactions`                          | `read_file`, `write_to_file`                                   |
| **Review L2**   | `User (as Reviewer)`  | `/feature review-design --level 2 {approve\|reject}` | `claude-3-sonnet`      | `Critical Analysis`, `Architectural Validation`                              | `read_file`                                                    |
| **Design L3**   | `Lead Engineer`       | `/feature design --level 3`                          | `claude-sonnet-4.5`    | `Detailed Design`, `Task Planning`                                           | `read_file`, `write_to_file`                                   |
| **Review L3**   | `User (as Reviewer)`  | `/feature review-design --level 3 {approve\|reject}` | `claude-3-sonnet`      | `Critical Analysis`, `Architectural Validation`                              | `read_file`                                                    |
| **Implement**   | `Developer`           | `/feature implement`                                 | `claude-sonnet-4.5`      | `Code Generation`, `Algorithmic Thinking`, `Task Planning`                     | `read_file`, `write_to_file`, `apply_diff`, `execute_command`  |
